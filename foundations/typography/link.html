Link Text Styles

Use link text styles for any text that can be interacted with to open something, like a modal, or navigate the user to a new place, like a different page or website.

Example

Link text is used for the “Link” component. It can also be used within paragraphs of text. Always pair with the “Link” text color.

Font Family

All heading texts use the Greycliff font family.

Styles

Text styles bind together multiple variables and use t-shirt naming to make distinguish between sizes.
