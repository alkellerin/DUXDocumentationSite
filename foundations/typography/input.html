Input Text Styles

Input text is the actual text that users type into a text field, such as a form field, search box, or any other interface element that accepts textual input.

Example

Input text styles are used for inputs like text fields, text areas, search bars, and dropdowns.

Font Family

All heading texts use the Greycliff font family.

Styles

Text styles bind together multiple variables and use t-shirt naming to make distinguish between sizes.
