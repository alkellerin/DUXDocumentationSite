Label Text Styles

Use label text styles for components with “labels” that only require a single-line of text.

Example

Label text is used with components like buttons, badges, and chips that require a single-line of text. Their fixed heights, which align with the 8px grid (ie 24px), allow their components to have consistant sizing.

Font Family

All heading texts use the Greycliff font family.

Styles

Text styles bind together multiple variables and use t-shirt naming to make distinguish between sizes.
