Spacing

Consistent and purposeful application of a spacing system fosters a more cohesive experience for users. It also establishes a foundation for responsive design and customizable UI density, improving the overall quality and accessibility of our products.

Spacing is applied to “padding” and “gap” properties when using auto layout in components like buttons.

Scale

The spacing system is built around a base unit of 8 pixels. This base unit determines the spacing scale and ensures visual consistency across products.
